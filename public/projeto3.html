Tipos e Formatos

O que é o Tipo de Formato e tipo de Ficheiro?

O tipo de ficheiro é a forma como os dados são organizados (ex: .txt ou .jpg). O formato especifica como esses dados são codificados e estruturados para que um software possa interpretá-los.

Tipo de Ficheiro

É uma categoria que o computador usa para identificar o conteúdo e com que programa pode abri-lo. Geralmente indicado pela extensão (ex: .docx, .jpg, .pdf).

Formato de Ficheiro

É a estrutura específica e a organização dos dados dentro de um ficheiro. Determina como a informação é codificada, armazenada e interpretada.

Características das Imagens Digitais

As características variam drasticamente dependendo do seu tipo fundamental:

I. Tipos Fundamentais

  • Raster (Bitmap): Composta por uma grelha de pixéis. Perde qualidade ao ser ampliada (ex: JPEG, PNG).
  • Vetorial: Composta por caminhos matemáticos (pontos, linhas, curvas). Pode ser ampliada infinitamente sem perder qualidade (ex: SVG, AI).

II. Resolução e Qualidade

  • Resolução: Quantidade de detalhe (geralmente em PPI - Pixels Per Inch).
  • Compressão:
    • Com Perdas (Lossy): Reduz o tamanho descartando dados (ex: JPEG).
    • Sem Perdas (Lossless): Preserva a qualidade original (ex: PNG).

III. Cor e Comportamento

  • Modelo de Cor:
    • RGB: Para ecrãs (luz).
    • CMYK: Para impressão (pigmento).
  • Transparência (Canal Alfa): Define a opacidade. Formatos como PNG e GIF suportam; JPEG não.
  • Extensões Comuns: JPG (compressão), PNG (transparência), SVG (vetorial).
